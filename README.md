# Microsoft SharePoint (microsoft-sharepoint)
Microsoft SharePoint provides collaboration and content management APIs including REST APIs, Microsoft Graph integration, webhooks, search, and the SharePoint Framework for building custom solutions.

## APIs

### SharePoint REST API
The SharePoint REST API provides direct access to SharePoint sites, lists, libraries, items, and other resources using OData endpoints.

### Microsoft Graph SharePoint Sites API
The Microsoft Graph SharePoint Sites API provides access to SharePoint sites, lists, and document libraries through the unified Microsoft Graph endpoint.

The SharePoint Webhooks API enables applications to subscribe to change notifications on SharePoint lists and libraries.

### SharePoint Framework (SPFx)
The SharePoint Framework is a development model for building client-side web parts, extensions, and adaptive card extensions for SharePoint and Microsoft Teams.
